# AWS-Static-Website-Deployment
# AWS S3 Static Website with Terraform & GitLab CI/CD

## Overview

This project demonstrates how to deploy and manage a static website on AWS using Amazon S3, Terraform Infrastructure as Code (IaC), and GitLab CI/CD pipelines.

The goal of this lab was to automate infrastructure deployment, apply Infrastructure as Code best practices, and gain hands-on experience with cloud automation workflows.

---

## Technologies Used

* AWS S3
* Terraform
* GitLab CI/CD
* Git
* Infrastructure as Code (IaC)
* AWS IAM

---

## Architecture

```text
Developer
    │
    ▼
GitLab Repository
    │
    ▼
GitLab CI/CD Pipeline
    │
    ▼
Terraform
    │
    ▼
Amazon S3 Bucket
    │
    ▼
Static Website Hosting
```

## Objectives
